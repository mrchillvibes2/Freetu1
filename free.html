HTTP Injector Tutorial

Posted by FreePre Team

HTTP Injector is a popular SSH tunneling app for Android. Here's how to set it up for unlimited HTTPS:

  1. Download & Install: Download from Google Play or APKPure. Enable "Unknown Sources" in settings.
  2. Import Config File: Get .ehi config files from SSH servers like SSHOcean. Tap "+" > File > Import .ehi file.
  3. Configure Settings:
    • SSH Host: Your server IP
    • SSH Port: 443
    • SNI Bug: Use a valid HTTPS domain
    • Payload: GET https://[host]/ HTTP/1.1[crlf]Host: [host][crlf]...
  4. Start Connection: Tap "Start". Check logs for "Connected" status.

Advanced Tips: Use SNI bug hosts for zero-rated HTTPS sites.

HTTP Custom Tutorial

Posted by FreePre Team

HTTP Custom is a lightweight app for unlimited HTTPS tunneling:

  1. Download HTTP Custom: Get from Google Play or APKPure.
  2. Import HC Config: Download .hc config files. Go to Import > Select .hc file.
  3. SSH Settings:
    • Server: SSH server IP
    • Port: 443
    • Proxy: Enable HTTPS proxy
  4. Connect: Tap "Connect". Monitor connection status.

Features: Supports HTTPS proxy customization.

Dark Tunnel Tutorial

Posted by FreePre Team

Dark Tunnel enables unlimited HTTPS with its modern features:

  1. Install Dark Tunnel: Download from Google Play or APKPure.
  2. Add SSH Account: Go to Accounts > Add SSH > Enter server details.
  3. Configure Tunnel:
    • Mode: SSH+Proxy
    • Proxy Type: HTTPS
    • SNI: Enable for HTTPS
  4. Connect & Test: Select account > Connect.

Features: Multi-protocol support for HTTPS.

ZivPN Tutorial

Posted by FreePre Team

ZivPN offers user-friendly unlimited HTTPS setup:

  1. Install ZivPN: Download from APKPure.
  2. Import Config: Obtain .ziv config files. Import via "Add Config."
  3. Configure Settings:
    • Server: SSH server IP
    • Port: 443
    • Protocol: HTTPS
  4. Start Connection: Tap "Connect" and verify.

Features: Simple HTTPS configuration.
